## About ASIX

Persiapan ENV:
ubah db connection sesuai dengan yang dipakai
contoh:
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=asix
DB_USERNAME=root
DB_PASSWORD=

Cara Penggunaan:
1. composer install
2. php artisan migrate
3. php artisan db:seed --class=UserSeeder
4. npm install
5. npm run dev
6. php artisan serve
